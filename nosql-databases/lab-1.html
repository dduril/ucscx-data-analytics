Lab 1 - MongoDB Queries

Start mongod server

C:\development\mongodb> mongod --dbpath=data

Import US Data from zips.json file

C:\Program Files\MongoDB\Server\3.2\bin> mongoimport
--db usdata
--collection cityinfo
--drop
--file C:\development\mongoDB\zips.json

Start mongo shell

> mongo

Explore databases

> show dbs
> use usdata
> show collections
