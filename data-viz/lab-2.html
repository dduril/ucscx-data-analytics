Tableau - Assignment #2

Report 1: KPI

  • Show Good, Average, and Poor Sales
  • Dimensions: Category, Region
  • KPI to be created based on the Order Quantity Column.
  • Create a Calculated Field and name in it "Quantity KPI"
  • Use Sum Aggregation for Quantity.
  • For KPI symbols, select "Shape" from the Marks drop-down menu.

Report 2: Filtering

  • Fields: Segment, Category
  • Dimensions: State
  • Measures: Sales
  • Labels: State, Sales

Report 3: Hierarchy

  • Create a Hierarchy named "Region Hierarchy" or "Location Hierarchy"
  • Dimensions: Region > State > Postal Code
  • Display the Hierarchy in the report for Measure: Sales.
