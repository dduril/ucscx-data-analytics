The Hadoop platform including MapReduce frameworks (MapReduce 2 and Spark), Hadoop Distributed File System (HDFS), development of data processing applications, data warehousing, Pig, Hive, SQL Processing (SparkSQL) and machine learning, distributed scalable NoSQL databases (Cassandra), and the distributed processing coordination system Zookeeper. The Cloudera Hadoop Platform and Java were used to complete the hands-on labs and programming assignments for this course.
