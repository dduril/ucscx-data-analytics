Focus on data analysis workflow - collecting, cleaning, manipulating, exploring, modeling and presenting data. Primary tools used include spreadsheet programs like Excel and Google Sheets to review and clean data and R/RStudio for data exploration and presentation. Popular packages from the R community and R Markdown for developing and sharing data analysis project work.

Additional topics include contrasting R and Python, how to find datasets for learning and development, common data formats (csv, json), file management, version control and best practices.
