Focus on the data analysis workflow - acquisition, parsing, manipulation, and presentation of datasets. Primary tools used include MS Excel and Google Sheets to review raw data and R and RStudio for data exploration and presentation. Popular packages from the R community including graphing and visualization tools for presentation of findings and results. Use of the R Markdown extension to present both R source code and output in an easy to view HTML format.

Additional topics include contrasting R with Python, how to find publicly available datasets for exploration and development, popular data formats (csv, tsv, txt, json), plots and graphing, file management, version control, documentation, best practices, and recommended resources.
