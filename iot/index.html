The Internet of Things: Big Data Processing and Analytics

  1. Home
  2. IoT

This course covers the Internet of Things (IoT), Industrial Internet of Things (IIoT), and Machine-to-Machine (M2M) data processing and analytics by using Amazon Web Services (AWS) and simulated data from devices, sensors, appliances and other data-collecting technology. Big data - Apache Spark, Kafka, Zookeeper combined with popular programming languages - Python and PySpark to process data in near real-time and extract meaning and insight.
